Hammer Readme
=============

You can find information on how to use Hammer on the project wiki:

http://wiki.github.com/benwazza/hammer

Other sources of information are:

* INSTALL.txt contains installation instructions

* The 'sample' directory included in the binary distribution contains
  a small build to illustrate the basic concepts of a Hammer build

* The src distribution provides lots of examples, with builds in
  src/demo and Hammer's own build in src/self

  http://github.com/benwazza/hammer

* DEV_README.txt in the src distribution covers basic development
  related information
